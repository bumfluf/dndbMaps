Privacy Policy

1. Introduction

D&D Beyond Maps Explorer does not collect personal data for advertising or tracking purposes. This privacy policy explains what information the extension reads, stores, and uses to provide its functionality.

2. Data Collected

The extension may process the following user-provided information:

  • D&D Beyond character names shown on the character sheet page
  • Google Drive folder URLs or folder IDs entered by the user
  • Cached map metadata and folder parsing state stored locally in the browser

3. How Data Is Collected

Data is collected only through direct user interaction and extension behavior:

  • The extension Settings page saves character-to-folder mappings when you add or update a mapping.
  • The content script reads the active D&D Beyond page to detect the character name for matching.
  • The background/content scripts fetch and cache Google Drive folder HTML when loading maps.

4. Purpose of Collection

Data is used only to:

  • Match the current D&D Beyond character page to the correct Google Drive folder
  • Load and display maps from the user-specified Drive folder
  • Cache Drive folder results for faster map loading
  • Autofill character names in the extension Settings page when a D&D Beyond tab is open

5. Data Storage

Data is stored locally in the browser using Chrome’s extension storage. Character-to-folder mappings and cached folder state are kept within the browser so the extension can restore settings and load maps more quickly. The extension does not store this data on any external server as part of normal operation.

6. Data Sharing

The extension does not share user data with third parties.

  • No analytics providers
  • No advertising networks
  • No external data brokers

7. Remote Code and External Access

The extension does not execute remote JavaScript or WebAssembly from external hosts. All extension code is packaged inside the extension bundle.

8. Security

Stored data is kept inside Chrome’s extension storage. The extension does not transmit user data to any external server under normal operation; it only makes browser fetch requests to Google Drive when loading user-specified maps.

9. User Control

You can delete or modify mappings at any time via the extension Settings page. Clearing browser storage or uninstalling the extension removes stored mappings and cached data.

10. Changes to this Policy

If the extension’s behavior changes, this policy will be updated accordingly. Always refer to the latest published version for the current data handling practices.
